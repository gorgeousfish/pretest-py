# pretest Examples

Runnable tutorials demonstrating the conditional extrapolation pre-test framework.

## Prerequisites

```bash
pip install pretest-did[data]     # pandas support
pip install matplotlib           # for plotting examples
```

## Tutorials

| File | Description | Level |
|------|-------------|-------|
| 01_quickstart.py | Minimal working example | Beginner |
| 02_prop99_replication.py | Replicate paper Prop 99 results | Intermediate |
| 03_advanced_pipeline.py | Step-by-step low-level API | Advanced |
| 04_event_study_plot.py | Plot customization showcase | Intermediate |

## Running

```bash
python examples/01_quickstart.py
python examples/02_prop99_replication.py
python examples/03_advanced_pipeline.py
python examples/04_event_study_plot.py
```

## Output

All generated figures are saved to `examples/output/`.
